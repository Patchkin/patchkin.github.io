Tic Tac Toe- The Game

The aim of this project is to create an online Tic-tac-toe application, in which players will be able to make moves on a 3x3 square grid. The first player to hit 3 consecutive marks in a row wins, or if both players fill up the grid without placing 3 marks in a row then it will be a tie.
